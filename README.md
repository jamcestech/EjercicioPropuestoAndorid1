# EjercicioPropuestoAndorid1
¡Adivina el número!

Para poner en práctica lo aprendido se os propone crear un aplicación desde cero que tenga un campo de texto donde el usuario pueda introducir un número del 1 al 10 y un botón para ver si ha acertado.

En el código Java compararemos el número introducido con un número que tengamos guardado, si es el mismo aparecerá el texto “¡Acertado!” en verde y si no, “Vuelve a intentarlo” en rojo.

Nota importante: No es necesario colocar los elementos adecuadamente en la pantalla, eso se explica en la siguiente subsección.
